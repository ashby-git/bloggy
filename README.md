### Description

A NextJS website where you can view blog posts and make comments.

### Goal

The main goal of this project was to make a blog using a CMS, with a responsive design and modern technology stack.

### Technology Stack / Tools

- Next.js
- Typescript
- Tailwind CSS
- Sanity CMS
- React Hooks
